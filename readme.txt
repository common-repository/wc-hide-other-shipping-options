== Description ==

Plugin will Hide Other Shipping Options when Free Shipping is available in woocommerce.

After Installing this plugin, it will Hide Other Shipping Options when Free Shipping is available in woocommerce. You can contact me on <a href="http://www.gagantiwari.info/contact/">My Blog</a> for further support.

Plugin Features

1- Hide Other Shipping Options when Free Shipping is available in woocommerce.

2- Only Free shiiping will be visible.

3- Hide All Other Shipping Options accept free shipping.

4- Fully Tested with number of sites.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wc-hide-other-shipping-options` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You are done.
